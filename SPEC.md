# 99乘法表闯关游戏 - 规格说明书

## 1. 项目概述

**项目名称**: 乘法小英雄  
**项目类型**: 手机端网页小游戏  
**核心功能**: 玩家通过解答99乘法表题目进行闯关，每关击败一个妖怪，全部通关后获得大奖杯  
**目标用户**: 6-12岁儿童学习乘法的小游戏

---

## 2. UI/UX 规格

### 2.1 布局结构

**页面组成**:
- **开始界面**: 游戏标题 + 开始按钮 + 最高记录显示
- **闯关界面**: 妖怪展示区 + 题目区域 + 答案输入区 + 进度显示
- **胜利/失败界面**: 结果展示 + 继续/重试按钮
- **通关界面**: 大奖杯动画 + 庆祝特效 + 重新开始按钮

**响应式设计**:
- 移动端优先设计 (375px - 428px)
- 最大宽度限制: 480px (居中显示)
- 全屏高度适配

## 3. 功能规格

### 3.1 游戏流程

```
开始界面 → 第1关 → 第2关 → ... → 第9关 → 通关界面
              ↓(答错)
           重新挑战当前关
```

### 3.2 核心功能

**关卡设计 (普通模式)**:
- 共9关，对应9的乘法 (1x1到9x9)
- 每关3道题目，需全部答对才能过关
- 题目随机从对应关卡的乘法式中选取
- 答错1题则本关失败，需要重新挑战

**题目生成规则**:
- 第1关: 1的乘法 (1x1 ~ 1x9)
- 第2关: 2的乘法 (2x1 ~ 2x9)
- ...
- 第9关: 9的乘法 (9x1 ~ 9x9)
- 每关3道题，随机抽取

**妖怪系统**:
- 共9个妖怪，对应9个关卡
- 妖怪名称: 小妖 → 中妖 → 大妖 → ... →  Boss
- 闯关成功时妖怪播放阵亡动画
- 妖怪按顺序逐步展示

**答案输入**:
- 数字键盘输入
- 支持直接点击答案候选 (4个选项)
- 提交按钮确认

**胜利条件**:
- 通关第9关且9个妖怪全部击败
- 显示大奖杯动画

### 3.3 数据处理

**游戏状态**:
```javascript
{
  currentLevel: 1,        // 当前关卡 (1-9)
  currentQuestion: 1,     // 当前题目序号 (1-3)
  score: 0,               // 答对题数
  defeatedMonsters: [],   // 已击败的妖怪数组
  gameStatus: 'start'     // start/playing/won/lost/victory
}
```

### 3.4 边界情况

- 用户未输入答案时禁用提交
- 输入非数字时自动过滤
- 答案错误时显示正确答案
- 全部通关后显示最终成绩

- [ ] 游戏可以正常启动，显示开始界面

- 移动端界面友好，操作流畅
- 妖怪动画流畅自然

- 颜色搭配协调
