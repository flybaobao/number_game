# 验收清单 - 乘法小英雄游戏

## 功能验收

### 开始界面
- [ ] 游戏标题正确显示 "乘法小英雄"
- [ ] 开始游戏按钮可见且可点击
- [ ] 点击开始按钮后进入第1关

### 闯关界面
- [ ] 显示当前关卡 (第X关)
- [ ] 显示当前题目序号 (第Y/3题)
- [ ] 妖怪图片/形象正确显示
- [ ] 题目格式正确显示 (如: 2 × 3 = ?)
- [ ] 答案输入区域可用
- [ ] 4个候选答案按钮显示正确
- [ ] 提交按钮可用

### 答题逻辑
- [ ] 正确答案显示绿色反馈 + 动画
- [ ] 错误答案显示红色反馈 + 抖动动画
- [ ] 错误后显示正确答案
- [ ] 答对后自动进入下一题或下一关

### 妖怪系统
- [ ] 每关对应一个妖怪
- [ ] 闯关成功时妖怪播放阵亡动画
- [ ] 妖怪按顺序逐步解锁

### 结果界面
- [ ] 闯关成功显示 "闯关成功!"
- [ ] 闯关失败显示 "闯关失败" + 重试按钮
- [ ] 继续按钮正常工作

### 通关界面
- [ ] 大奖杯正确显示
- [ ] 奖杯发光动画播放
- [ ] 庆祝特效可见
- [ ] 重新开始按钮可用
